Please Don’t!

“Every negative event contains within it the seed of an equal or greater benefit.” – Napoleon

So let’s see if you got it right. Do you remember how to do negative te form?

Positive: たべて

Negative: たべないで

Simply put, all you need to do is add 下さい to the negative te-form verb. How did you do? If you didn’t get it wrong, just be sure to think through the logic of this way of doing it. If you got it right, well, then you’re awesome opossum.

Let’s take a look at a few examples.

a あれ　を　たべないで下さい
Please don’t eat that (over there).

a こういち君　の　いえ　に　行かないで下さい。
Please don’t go to Koichi’s home.

a ゲーム　を　かわないでくださいね。
Please don’t buy a game, alright?

a 見ないで下さい！
Please don’t look!

So as long as you know your negative te-form, you should know this as well (because you know basic 下さい already too!). Let’s give it some practice anyways, though.
