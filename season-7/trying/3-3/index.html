Trying Verbs

“I can accept failure, everyone fails at something. But I can’t accept not trying.” – Michael Jordan

Let’s give that pattern a try (har har). Go ahead and change these verbs into “trying” (てみる) form.

ならう Convert →

a ならってみる
Try to learn

のむ Convert →

a のんでみる
Try to drink

かう Convert →

a かってみる
Try to buy

いそぐ Convert →

a いそいでみる
Try to hurry

入る Convert →

a 入ってみる
Try to enter

かえる Convert →

a かえってみる
Try to return home

する Convert →

a してみる
Try to do

よむ Convert →

a よんでみる
Try to read

かく Convert →

a かいてみる
Try to write

 See how easy this is as long as you know your te-form? Once you’re able to get through these, move on to the next page. If you’re having trouble, make sure you review te-form and how it works, it’ll make things easier for you as you move forward. Let’s put these in real sentences, now!
