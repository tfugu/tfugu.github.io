The これ, それ, あれ Quiz

“Cheating on a quiz show? That’s sort of like plagiarizing a comic strip.” - Paul Scofield

Now it’s time to mix things up a little bit. Instead of having you do entire sentences, or translating things, I just want you to fill in the blank. The blank, of course, is going to be either これ, それ, or あれ. Up to the challenge? I think it’ll help you figure out which one to use when a lot better. Let’s give it a shot.

You: ____　は　あさごはん　ですか？　Is that breakfast? Answer →

それ, which refers to something that the person the speaker is talking to has.

Person B: はい、____　は　あさごはん　です。Yes, this is breakfast. Answer →

これ, which refers to something the speaker has in their possession.

You: ____　は　いす　ですか？　Is this a chair? Answer →

これ

Person B: いいえ、____　は　いす　じゃありません。　No, that is not a chair. Answer →

それ

You: ____　は　じてんしゃ　ですか？　Is that over there a bike? Answer →

あれ, which refers to something away from both the speaker and the person being spoken to

Person B: はい、_____　は　じてんしゃ　です。Answer →

あれ, you can base this answer off the previous answer (previous fill-in-the-blank), since that one was あれ (away from both people) as well.

Person B: ____　は　のみもの　です。This is a beverage. Answer →

これ

You: _____　は　のみもの　ですか？ Is that a beverage? Answer →

それ

Person B: はい、____　は　のみもの　です。Yes, this is a beverage. Answer →

これ

You: いいえ、____　は　のみもの　じゃありません。____ は　のみもの　です。No, that is not a beverage. That (over there) is a beverage. Answer →

1. それ　2. あれ

What are the differences between これ, それ, and あれ? Make sure you can explain the differences (possibly out loud?) before moving on to the next page. In theory, you should have these down pretty well by now, if not, go back and review!
