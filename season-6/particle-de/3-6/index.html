で Practice Two

“If you don’t risk anything, you risk even more.” – Erica Jong

Let’s take a look at です #2 – once again, you’re going to finish the sentence. I’ll give you a prompt containing something that you could  do something else with. For example:

コンピューターで ... →

aコンピューターで　この　レポート　を　かいた。
Via a computer I wrote this report / I wrote this report using a computer

I’ll provide example answers for you, but come up with your own answers and compare. If you’re not sure about any of them, put them on Lang-8 and get them checked.

耳で ... →

a耳で　シークレット　を　きいた。
Using my ear(s) I heard a secret.

血で ... →

a血で　この　え　を　かいた。
I drew this painting using blood

休みで ... →

a休みで　自分　を　リチャージ　しました。
Using my break/vacation I recharged myself.

日光で ... →

a日光で　レーズン　を　つくった。
Using the sunlight I made raisins.

くるまで ... →

aくるまで　とうきょう　に　いった。
I went to Tokyo via car.

ちかてつで ... →

aちかてつで　かえった。
I returned (home) via subway.

肉で ... →

a肉で　ヘルメット　を　つくった。
I made a helmet using meat.

How did that go for you? Were you able to answer the questions? All the words were from previous things, so hopefully they helped you review, too.

When you’re able to finish all the sentences, move on to the next で which means “Because Of.”

By finishing this page, you’ve solidified the second type of で, and hopefully learned the differences between で number one and two as well. There’s two more で particles to go, though, so make sure you keep them separate!
