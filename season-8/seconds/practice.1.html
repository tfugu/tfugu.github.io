Practice

“As if you could kill time without injuring eternity” – Henry David Thoreau

We’ll keep this one short so you can wrap all this “time” stuff up. Seconds are also short, after all.

Exercise 1

Make sure you’re able to say and read all of these “second-words.”

a ３秒 Translate →

さんびょう

a ４秒 Translate →

よんびょう

a ８秒 Translate →

はちびょう

a ３５秒 Translate →

さんじゅうごびょう

a ３０秒 Translate →

さんじゅうびょう

a ４５秒 Translate →

よんじゅうごびょう

a １００秒 Translate →

ひゃくびょう

a ２０秒 Translate →

にじゅうびょう

a １０秒 Translate →

じゅうびょう

Those shouldn’t be too hard for you, so let’s look at some sentences.

Exercise 2

Make sure you can read and understand all of these.

a １分は６０秒です。Translate →

1 minute is 60 seconds.

a 約１０秒かかりました。Translate →

It’s about 10 seconds.

後３０秒したらこの部屋から出て下さい。Translate →

After 30 seconds please leave this room.

a １分半待って下さい。Translate →

Please wait one and a half minutes.

Come up with variations on these sentences as well. There’s no Lang-8 post for this practice page since sentences are so short (that shouldn’t stop you from writing one even if it’s not on this topic) so play around with what you have here before moving on to the last chapter before the review.
