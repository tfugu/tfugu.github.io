6 Stroke Kanji

“If you will make the necessary effort you can develop any talent.” – Wallace D Wattles

We’re going to up the number of kanji per set now to 15. I think you’re getting better at learning kanji, and it’s always good to push and challenge yourself. Don’t think of how difficult this will be (it won’t be that bad, jeeeeesh). Instead, think of how luck you are to get to learn five extra kanji per group. That’s awesome, if you ask me.

Still working on this page – if the kanji has a link, that means the kanji page is done. I’ll be posting up updates pretty often – if you want to know what those updates are, make sure to follow TextFugu on Twitter.
